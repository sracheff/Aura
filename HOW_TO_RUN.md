# LUMA — How to Run the App

## Quick Start

Open Terminal, then run:

```bash
cd "/Users/stephenracheff/Documents/Claude/Projects/Salon All-in One/luma-app"
npm install
npm run dev
```

Then open your browser to: **http://localhost:3000**

---

## What's Included

| Route | Page |
|-------|------|
| `/` | Landing / Marketing page |
| `/dashboard` | Main dashboard with KPIs, charts, schedule |
| `/calendar` | Week/day/list calendar with appointment detail |
| `/clients` | Client CRM with loyalty tiers & profiles |
| `/pos` | Point of Sale — checkout with cart |
| `/finance` | Revenue, expenses, profit charts |
| `/staff` | Stylist performance & commissions |
| `/referrals` | Referral campaigns & advocate leaderboard |
| `/inventory` | Product stock with low-stock alerts |
| `/reports` | Full analytics & export |
| `/settings` | Salon config, notifications, AI features |

## Deploy to Vercel (1-click)

1. Push this folder to a GitHub repo
2. Go to vercel.com → Import Project
3. Select the repo → Deploy

That's it — live in under 2 minutes.
